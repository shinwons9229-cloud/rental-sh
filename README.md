# 시헌테크 - AS 매뉴얼 시스템

프린터 AS 매뉴얼을 관리하고 고객에게 제공하는 웹 기반 시스템입니다. 토스 디자인 시스템을 기반으로 깔끔하고 직관적인 UI를 제공합니다.

## 프로젝트 개요

시헌테크는 프린터 관련 AS 매뉴얼을 효율적으로 관리하고 고객들에게 쉽게 제공하기 위한 웹 애플리케이션입니다. 공지사항 시스템과 유사한 구조로, 관리자가 매뉴얼을 작성하면 고객들이 검색하고 열람할 수 있습니다.

## 도메인 정보

- **운영 도메인**: rental-sh.com

## 주요 기능

### 현재 구현된 기능

#### 1. 고객용 페이지 (index.html)
- 매뉴얼 리스트 조회
  - 카드 형태의 직관적인 UI
  - 카테고리별 필터링 (레이저 프린터, 잉크젯 프린터, 복합기, 기타)
  - 제목 및 태그 기반 검색
  - 페이지네이션
- 매뉴얼 상세보기
  - 모달 형태로 전체 내용 표시
  - 조회수 자동 카운팅
  - 태그 표시
- 반응형 디자인
  - 모바일, 태블릿, 데스크톱 대응

#### 2. 관리자 페이지 (admin.html)
- 매뉴얼 작성 및 관리
  - 제목, 카테고리, 내용, 태그 입력
  - 공개/비공개 설정
  - 실시간 CRUD 작업 (생성, 수정, 삭제)
- 매뉴얼 리스트 관리
  - 테이블 형태의 관리 화면
  - 카테고리별 필터링
  - 조회수, 공개여부 확인
  - 작성일 표시
- 수정 및 삭제 기능
  - 인라인 수정 기능
  - 삭제 확인 모달

#### 3. 디자인 시스템
- 토스 디자인 가이드 기반
  - 깔끔한 컬러 팔레트
  - 일관된 타이포그래피
  - 부드러운 인터랙션
- Pretendard 폰트 사용
- 반응형 레이아웃

## 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **폰트**: Pretendard (Google Fonts)
- **디자인**: 토스 디자인 시스템
- **데이터 저장**: RESTful Table API
- **아키텍처**: SPA (Single Page Application)

## 데이터 모델

### manuals 테이블

| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID (자동 생성) |
| title | text | 매뉴얼 제목 |
| category | text | 카테고리 (레이저 프린터, 잉크젯 프린터, 복합기, 기타) |
| content | rich_text | 매뉴얼 본문 내용 |
| tags | text | 검색용 태그 (쉼표로 구분) |
| views | number | 조회수 |
| is_published | bool | 공개 여부 |
| created_at | datetime | 작성일 |

## 사용 방법

### 고객 페이지 사용

1. **메인 페이지 접속**: index.html로 접속
2. **매뉴얼 검색**:
   - 상단 검색창에 키워드 입력 (제목, 태그 검색 가능)
   - 카테고리 버튼으로 필터링
3. **매뉴얼 보기**:
   - 카드를 클릭하면 상세 내용 모달 표시
   - 조회수 자동 증가

### 관리자 페이지 사용

1. **관리자 페이지 접속**: admin.html로 접속
2. **매뉴얼 작성**:
   - 제목, 카테고리, 내용 입력
   - 태그를 쉼표(,)로 구분하여 입력
   - "즉시 공개" 체크박스로 공개 여부 설정
   - "등록하기" 버튼 클릭
3. **매뉴얼 수정**:
   - 리스트에서 "수정" 버튼 클릭
   - 폼에 기존 데이터가 자동 입력됨
   - 수정 후 "수정하기" 버튼 클릭
4. **매뉴얼 삭제**:
   - 리스트에서 "삭제" 버튼 클릭
   - 확인 모달에서 "삭제" 버튼 클릭

## API 엔드포인트

모든 API는 RESTful Table API를 사용합니다.

### 매뉴얼 조회
```
GET /tables/manuals?page=1&limit=10&sort=-created_at
```

### 매뉴얼 상세
```
GET /tables/manuals/{id}
```

### 매뉴얼 생성
```
POST /tables/manuals
Content-Type: application/json

{
  "title": "매뉴얼 제목",
  "category": "레이저 프린터",
  "content": "매뉴얼 내용",
  "tags": "태그1, 태그2",
  "is_published": true,
  "views": 0
}
```

### 매뉴얼 수정
```
PUT /tables/manuals/{id}
Content-Type: application/json

{
  "title": "수정된 제목",
  "category": "잉크젯 프린터",
  ...
}
```

### 매뉴얼 삭제
```
DELETE /tables/manuals/{id}
```

### 조회수 증가
```
PATCH /tables/manuals/{id}
Content-Type: application/json

{
  "views": 현재조회수 + 1
}
```
